
# 50. Pow(x, n)  

**Difficulty:** Medium  

You are given a **floating-point number** `x` and an **integer** `n`.  

Your task is to calculate `x` raised to the power `n`, i.e., `x^n`.  

You **must** implement this function in `O(log n)` time complexity.  

---  

### Example 1:  
**Input:**  
`x = 2.00000`, `n = 10`  

**Output:**  
`1024.00000`  

**Explanation:**  
- \(2^{10} = 1024\).  

---  

### Example 2:  
**Input:**  
`x = 2.10000`, `n = 3`  

**Output:**  
`9.26100`  

**Explanation:**  
- \(2.1^3 = 9.261\).  

---  

### Example 3:  
**Input:**  
`x = 2.00000`, `n = -2`  

**Output:**  
`0.25000`  

**Explanation:**  
- \(2^{-2} = \frac{1}{2^2} = \frac{1}{4} = 0.25\).
